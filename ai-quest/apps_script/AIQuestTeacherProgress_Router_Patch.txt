CSAI2102 AI QUEST — TEACHER PROGRESS ROUTER PATCH v1.1

1) เพิ่มไฟล์ AIQuestTeacherProgress_Receiver.gs เข้า Apps Script โปรเจกต์เดียวกับ Code.gs และ AIQuestFlow_Receiver.gs

2) เปิด Code.gs และหา function doPost(e)

3) หลัง parse payload แล้ว และก่อน Router ทั่วไป ให้เพิ่ม:

// =====================================================
// AI QUEST TEACHER PROGRESS ROUTER v1.1
// Multi-student class overview, detail and diagnostics
// =====================================================
if (
  String(payload.module || '').trim().toUpperCase() === 'AIQTEACHER' ||
  String(payload.action || '').trim().toUpperCase() === 'PING' ||
  String(payload.action || '').trim().toUpperCase() === 'TEACHER_PING' ||
  String(payload.action || '').trim().toUpperCase() === 'GET_CLASS_OVERVIEW' ||
  String(payload.action || '').trim().toUpperCase() === 'GET_STUDENT_DETAIL'
) {
  if (
    typeof AIQTEACHER === 'undefined' ||
    typeof AIQTEACHER.handle !== 'function'
  ) {
    return jsonOut({
      ok: false,
      code: 'AIQTEACHER_MODULE_MISSING',
      message: 'ยังไม่ได้เพิ่ม AIQuestTeacherProgress_Receiver.gs หรือ deployment ยังเป็นเวอร์ชันเก่า',
      action: payload.action || '',
      module: payload.module || ''
    });
  }
  return jsonOut(AIQTEACHER.handle(payload));
}

- ต้องมี AIQuestFlow_Receiver.gs เพราะ AIQTEACHER ใช้ฟังก์ชันอ่าน Mission/Coding/Reflection จาก AIQFLOW
- ห้ามสร้าง doPost(e) ใหม่
- ต้องวาง Router นี้ก่อน Router ทั่วไป มิฉะนั้น request อาจถูก Router อื่นรับไปก่อน
- Save ทุกไฟล์
- Deploy > Manage deployments > Edit > Version: New version > Deploy
- ใช้ URL ที่ลงท้าย /exec เดิมได้ แต่ต้องสร้าง New version จริง

รหัสวินิจฉัย
- AIQTEACHER_MODULE_MISSING = ไม่มี Receiver หรือยังไม่ได้ Deploy เวอร์ชันใหม่
- AIQFLOW_DEPENDENCY_MISSING = ไม่มี/เก่าเกินไป AIQuestFlow_Receiver.gs
- SPREADSHEET_NOT_FOUND = Apps Script ไม่ได้ผูกกับ Google Sheet ที่ถูกต้อง
- UNKNOWN_TEACHER_ACTION = Router ส่งถึงโมดูลแล้ว แต่ action ไม่รองรับ

Actions
- PING / TEACHER_PING
- GET_CLASS_OVERVIEW: รับ section, query
- GET_STUDENT_DETAIL: รับ section, studentId
