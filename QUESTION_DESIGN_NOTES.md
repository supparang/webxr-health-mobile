# Question Design Notes v1a

หลักปรับตัวเลือกคำตอบให้เหมาะกับ ป.ตรี:

1. หลีกเลี่ยงตัวเลือกผิดแบบตลก/หลุดบริบท เช่น lunch, emoji, whatever
2. ตัวเลือกผิดควรดู “เป็นไปได้” แต่ผิดเพราะเหตุผลทางทักษะ
3. Main idea ใช้ distractor 3 แบบ: too broad / too narrow / irrelevant
4. Academic tone ใช้ตัวเลือกที่ดูสุภาพบางส่วน แต่ยังไม่เป็น academic enough
5. Citation ใช้ตัวเลือกที่ paraphrase เหมือนถูก แต่ขาด citation หรือเปลี่ยนความหมาย
6. Data ใช้ตัวเลือกที่อ่าน trend ผิดเล็กน้อย ไม่ใช่ผิดชัดทันที
7. Practice ยังง่ายได้ แต่ Boss ควรตัวเลือกใกล้ขึ้นและจับเวลา
