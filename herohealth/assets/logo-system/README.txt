HeroHealth Logo System Pack

Folders
- logos/: hub logo, primary logo, icon logo, zone logos, and game logos
- badges/: reward and achievement badges
- hud/: UI HUD icons
- code/: ready-to-paste CSS and demo HTML

Quick use
1) Copy folders into /herohealth/assets/
2) Link code/herohealth-logo-system.css into your page
3) Use logos/hub-logo.svg in Hub v2 top bar
4) Use logos/*-logo.svg on launcher cards and result pages
5) Use hud/*.svg inside HUD pills
6) Use badges/*.svg on rewards, trophies, and summary pages

Recommended structure
/herohealth/assets/logo-system/
  logos/
  badges/
  hud/
  code/

Notes
- All assets are SVG for crisp scaling on PC, mobile, and VR-friendly UIs.
- Text is editable inside the SVG files.
