- เข้า run พร้อมกัน

วิธีใช้ Hydration Coop

โหมดนี้ไม่แข่งกัน แต่ช่วยกันทำเป้าของทีม

  • Host สร้างห้องก่อน แล้วส่ง room code ให้เพื่อน
  • เมื่อเข้าห้องแล้ว ให้ทั้งสองคนกด Ready
  • Host กด Start เพื่อพาเข้า run page ทั้งคู่
  • run page จะเริ่มพร้อมกันอีกชั้นด้วย run barrier
  • หลังจบรอบ ระบบจะรอครบ 2 คนแล้วสรุปผลแบบทีม
