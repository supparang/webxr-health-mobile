GOODJUNK • 4 MODES

เลือกโหมดการเล่น

โหมดเดี่ยวจะเป็น Phase Boss ส่วนโหมดหลายคนจะแยกเป็น Race, Battle, และ Coop แบบคนละ flow ชัดเจน

🟢 SOLO BOSS

Solo Boss

เล่นคนเดียวแบบเป็นด่าน Phase 1 → Phase 2 → Boss และสู้กับ Junk King

•มี weak spot / junk storm / enrage
•เหมาะกับโหมดฝึกเดี่ยวและเก็บ score
•ใช้หน้า run แยก ไม่ปน multiplayer
🔵 RACE

Race

แข่งกันทำคะแนน ใคร score สูงสุดชนะ เรียงอันดับตามคะแนน miss และ streak

•มี lobby / countdown / result board
•ทุกคนเริ่มพร้อมกันตาม start time
•เหมาะกับแข่งรายบุคคล
🟠 BATTLE

Battle

แข่งแบบปะทะกันโดยตรง มีจังหวะโจมตี-กดดันคู่แข่ง และผลแพ้ชนะชัดเจนกว่า race

•ควรมี attack / shield / pressure event
•สรุปผลเป็น winner / loser ชัดเจน
•ต้องใช้ run logic แยกจาก race
🟣 COOP

Coop

เล่นร่วมมือกัน เก็บคะแนนรวมของทีม ช่วยกันผ่านเป้าหมายเดียว ไม่เน้นแข่งกันเอง

•ใช้ team score / shared goal
•เหมาะกับการเล่นร่วมกันและช่วยกันเคลียร์ภารกิจ
•ต้องมี summary แบบทีม
ตอนนี้แยก logic เป็น 4 flow ชัดเจน: phaseboss, race, battle, coop เพื่อไม่ให้กติกาคนละโหมดชนกัน
