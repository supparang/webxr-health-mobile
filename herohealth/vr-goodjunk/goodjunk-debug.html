เช็กไฟล์ Multiplayer ให้ครบก่อน deploy

หน้านี้ใช้เช็กว่าไฟล์สำคัญของ GoodJunk มีอยู่จริงหรือไม่ พร้อมเปิด test scenario สำหรับ Solo, Race, Battle และ Coop ได้จากหน้าเดียว

สิ่งที่หน้า debug นี้ช่วยได้

  • เช็ก launcher / lobby / run / alias bridge
  • สรุปจำนวนไฟล์ที่ OK / Missing
  • เปิดลิงก์ test ได้ทันที
  • ช่วยไล่หาจุดที่อาจทำให้เกิด 404 ก่อนทดสอบหลายเครื่อง

ข้อจำกัด

  • เช็กได้ว่าไฟล์เปิดได้หรือไม่
  • ยังไม่ยืนยันว่า gameplay ข้างในไม่มี bug
  • ควรใช้ร่วมกับการเปิดเล่นจริงอย่างน้อย 1 รอบต่อโหมด
