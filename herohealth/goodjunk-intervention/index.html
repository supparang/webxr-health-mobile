ใช้หน้านี้เป็นจุดเริ่มต้นของ flow ทั้งหมดได้เลย: Teacher → Student → Pretest → Game → Posttest → Parent → Follow-up

Context ปัจจุบัน

กำลังโหลด...
หน้านี้จะอ่านข้อมูลจาก URL และจาก localStorage (`GJI_CTX`) แล้วรวมกันให้อัตโนมัติ

ลำดับ flow ที่แนะนำ

  1. Teacher Panel ตั้งค่า session และข้อมูลนักเรียน
  2. Student Launcher ตรวจข้อมูลก่อนเริ่ม
  3. Pre-Knowledge และ Pre-Behavior
  4. เข้าเกม GoodJunk Intervention
  5. Post-Knowledge → Post-Behavior → Post-Choice
  6. Completion
  7. Parent / Follow-up ตามแผนวิจัย

เข้าใช้งานตามบทบาท

Teacher
ตั้งค่า session, ข้อมูลนักเรียน, condition, difficulty และสร้างลิงก์ flow
Student
ตรวจข้อมูลก่อนเริ่มกิจกรรม และเข้าสู่ pretest หรือเข้าเกมตาม context ปัจจุบัน
Assessments
เข้าแบบประเมินก่อนเล่นหรือหลังเล่นแบบตรงหน้าได้ทันที
Parent & Follow-up
เข้าแบบสอบถามผู้ปกครอง หรือหน้าติดตามผลระยะสั้น / รายสัปดาห์

หมายเหตุการใช้งาน

  • ถ้าเริ่มจาก Teacher Panel ระบบจะส่ง context ไปทุกหน้าสะดวกที่สุด
  • ถ้าเข้าเกมตรง ควรมี session และ pid อยู่ใน context ก่อน
  • หน้า Post-Knowledge เป็นทางหลักหลังจบเกม
  • Parent Summary จะอ่านข้อมูลจากผลเกมและแบบสอบถามผู้ปกครอง
  • Follow-up จะโหลดคำตอบเดิมขึ้นมาได้ถ้าเคยบันทึกแล้ว
