ลำดับ flow ที่แนะนำ

1) Teacher Panel
ตั้งค่า session และข้อมูลนักเรียน
2) Student Launcher
ตรวจสอบข้อมูลก่อนเริ่มกิจกรรม
3) Pre-Knowledge → Pre-Behavior
ทำแบบประเมินก่อนเล่นเกม
4) Game → Posttest → Completion
เล่นเกมและจบ flow หลังเรียน
5) Parent / Follow-up / Tools
สรุปผล ติดตามผล และ export ข้อมูล

เข้าใช้งานตามบทบาท

Teacher

Student

Assessments
เข้า pretest, posttest และ completion ได้ทันที
Parent / Follow-up
เข้าแบบสอบถามผู้ปกครอง สรุปผล และติดตามผล

หมายเหตุการใช้งาน

  • ควรเริ่มจาก Teacher Panel ถ้าต้องการ context ครบที่สุด
  • ถ้าเข้าเกมตรง ควรมี session และ pid อยู่ใน context ก่อน
  • หลังจบเกม แนะนำให้ไป Post-Knowledge → Post-Behavior → Post-Choice → Completion
  • Parent Summary จะอ่านข้อมูลจากผลเกมและแบบสอบถามผู้ปกครอง
  • Follow-up จะโหลดคำตอบเดิมขึ้นมาได้ถ้าเคยบันทึกแล้ว
