⭐ SOLO MISSION

Solo

ผจญภัยคนเดียว เก็บอาหารดีให้ถึงเป้าหมาย แล้วไปสู้กับ Junk King
• เข้า Warmup ก่อนทุกครั้ง
• เล่นจบแล้วค่อยไป Cooldown
• ใช้ run: vr-goodjunk/goodjunk-vr.html
🏁 MULTIPLAYER

Race

แข่งคะแนนกับผู้เล่นคนอื่น ใครเก็บอาหารดีได้มากที่สุดชนะ
• เข้า lobby ก่อน
• host กด start
• ใช้ run แยกจาก solo
👯 DUET MODE

Duet

เล่นคู่ 2 คน ช่วยกันเก็บของดี ทำคะแนนของฉัน + คะแนนเพื่อน + คะแนนคู่
• เข้า Duet Lobby ก่อน
• ready ครบ 2/2 แล้ว host กด start
• ใช้ run แยกเฉพาะของ duet
⚔️ BATTLE

Battle
