GoodJunk Index

หน้าเมนูกลางสำหรับเปิดทุกหน้าของ GoodJunk classroom suite จากจุดเดียว ใช้ตั้งค่า room / view / diff / time ครั้งเดียว แล้วกดไปยัง Teacher, Board, Kiosk, Student Join, Launcher, Run page หรือหน้า Essential ได้ทันที
พร้อมใช้งาน
Classroom Control
หน้าสำหรับครูและการควบคุมห้องเรียน
👩‍🏫
Teacher Dashboard
หน้าควบคุมเต็ม ดู roster, attendance, policy, broadcast, macros, QR และ export
✨
Teacher Essential
เวอร์ชันย่อสำหรับใช้จริงหน้าห้อง เน้น Start/End class, QR, Board และ lock room
🧩
Launcher
หน้า launch รวมสำหรับเลือก Teacher, Board, Student Join, Run, Battle test และ Solo test
Board / Projector
หน้าสำหรับจอหน้าห้อง โปรเจกเตอร์ และ kiosk
📺
Live Board
กระดานคะแนนเต็ม มี auto cycle, announcement, winner spotlight และ room summary
🪄
Board Essential
เวอร์ชันเรียบมากสำหรับโปรเจกเตอร์ โฟกัส scoreboard, progress และ announcement
🖥️
Kiosk
หน้า kiosk สำหรับทีวี/โปรเจกเตอร์ มี clean mode, hide UI และ fullscreen friendly
Player Entry
หน้าสำหรับนักเรียนและลิงก์เข้าเล่น
🧑‍🎓
Student Join
ให้นักเรียนกรอกชื่อ / PID รับ room จากลิงก์ครู และเช็ก duplicate PID / room policy
⚔️
Run Game (Battle)
เปิด run page ตรงในโหมด battle สำหรับเดโม ทดสอบ compare table และ rematch
🥦
Run Game (Solo)
เปิด run page ตรงในโหมด solo สำหรับทดสอบ gameplay และ end summary แบบเดี่ยว

Suggested Flow
1) เปิด Teacher Essential หรือ Teacher Dashboard
2) เปิด Board Essential หรือ Kiosk บนจอหน้า class
3) ส่งลิงก์ Student Join หรือ QR ให้นักเรียน
4) เริ่มคาบด้วย Start Class Macro
5) จบคาบด้วย End Class Macro
ถ้าจะใช้งานแบบง่ายสุดในห้องจริง:
ครู → Teacher Essential
โปรเจกเตอร์ → Kiosk
นักเรียน → Student Join
