# PRE_FIREBASE_QA_LOCK_v1i

## เป้าหมาย
แพตช์นี้เป็นจุดล็อกคุณภาพก่อนย้ายไป Firebase

## เพิ่ม
1. QA Lock Center
2. Runtime Error Catcher
3. Question Bank Summary
4. LocalStorage Health
5. Export Full Backup JSON
6. Import Backup JSON
7. Export Firebase Preview JSON
8. Clear Active Run
9. Clear Runtime Errors
10. Browser/Device Debug

## ขั้นตอนแนะนำก่อนต่อ Firebase
1. เปิดเกมบน PC และ Mobile
2. เข้า QA Lock Center
3. ตรวจให้ checks ผ่านมากที่สุด
4. Export Full Backup JSON
5. เล่น Practice/Boss/Exam อย่างน้อย 1 รอบ
6. Export Firebase Preview JSON
7. ใช้ preview ออกแบบ Firestore collections/rules
