# STUDENT_SIMPLE_MODE_v1z4

## ปัญหา
ระบบครบมากแล้ว แต่อาจยาก/ซับซ้อนเกินไปสำหรับนักศึกษาครั้งแรก

## Fix
เพิ่ม Student Simple Mode เป็น default

## Simple Mode
- Default difficulty = Easy
- ซ่อน Teacher Tools, QA, Rubric, Revision, Class Activity
- แสดง Core Mission ก่อน
- Support Mission ปลดทีหลัง
- Boss Gate/Replay/Exam ปลดตาม progress
- มีปุ่ม Advanced สำหรับอาจารย์

## Unlock Level
Lv.1: Core Mission
Lv.2: Support Mission
Lv.3: Boss Gate / Replay / Exam
Lv.4: Revision visibility

## Teacher Advanced
อาจารย์กด Unlock Teacher Advanced Mode เพื่อเปิดระบบเต็ม
