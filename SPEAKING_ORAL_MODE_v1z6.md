# SPEAKING_ORAL_MODE_v1z6

## Problem
Speaking Mission ดูเหมือนให้พิมพ์ตอบ ทำให้สับสนและกลายเป็น Writing

## Fix
- Oral Task First
- Start Speaking timer
- I Finished Speaking
- Optional evidence notes/transcript
- I spoke checkbox
- Submit ต้องมี evidence ว่าพูดแล้ว
- Notes/transcript เป็นหลักฐานเสริม ไม่ใช่คำตอบหลัก

## Teaching logic
Speaking evidence = performance + self-check + optional transcript/notes
