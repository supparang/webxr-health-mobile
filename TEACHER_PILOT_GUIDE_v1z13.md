# TEACHER_PILOT_GUIDE_v1z13

## Before class
1. Upload index.html, eap-hero.js, eap-hero.css.
2. Use cache: ?x=v1z13
3. Open Pilot Readiness.
4. Reset Demo Data if needed.
5. Set Easy A2 or Normal B1.

## During class
1. Ask students to use Student Safe Start / Student Map.
2. Let students use AI Help as scaffold, not final answer.
3. Have students complete at least one evidence item.
4. For Speaking, check timer and notes.
5. For Listening, avoid showing transcript before submit.

## After class
1. Open Teacher Review Queue.
2. Filter by Skill / Session / Pending.
3. Review with rubric.
4. Export Portfolio CSV and Rubric CSV.

## Note
This version stores data in each browser using localStorage. For multi-device teacher dashboard, use Firebase in v2.
