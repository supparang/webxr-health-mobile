# 02 — คู่มือครู 1 หน้า

## ก่อนเริ่มคาบ

1. เปิด `teacher.html`
2. ไปที่ `Start Class`
3. ใส่ Session Code เช่น `VOCAB-WEEK3`
4. เลือก Section / Bank / Mode / Difficulty
5. กดสร้าง Join Link
6. Copy link ส่งให้นักศึกษาใน Classroom / Line / LMS

## ระหว่างคาบ

1. กด `Update Dashboard`
2. ดูกลุ่ม Need Help / Needs Review / On Track / Advanced
3. ดู Top Weak Terms
4. คลิกดูรายบุคคลเฉพาะกลุ่ม Need Help / Review

## ท้ายคาบ

1. ไปที่ `End Class`
2. กด `Update End Class`
3. ตรวจ Active Students / Missing Students / Top Weak Terms / What to Teach Next
4. Copy Summary
5. กด Archive Report ถ้าต้องการเก็บหลักฐาน

## รายสัปดาห์

1. ไปที่ `Weekly Report`
2. ใส่ Week Code เช่น `VOCAB-WEEK3`
3. Build Weekly Report
4. Copy Weekly Report
5. Archive Weekly Report

## Feedback

1. ไปที่ `Feedback Center`
2. เลือก Target Group เช่น `Need Help + Review`
3. Build Feedback
4. Copy Feedback Text
5. หรือ Create Follow-up Rows เพื่อบันทึกลง `teacher_followup`
