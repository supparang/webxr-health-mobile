# LISTENING_TRANSCRIPT_CONTROL_v1u

## ปัญหาที่แก้
Listening Mission ไม่ควรโชว์ full script ก่อนฟัง เพราะจะกลายเป็น Reading Mission

## Flow ใหม่
1. ก่อนทำ: แสดง Topic/Goal เท่านั้น
2. ระหว่างทำ: Play Lecture และจด notes
3. ขอช่วยได้: Transcript Hint แสดงเฉพาะต้น script
4. ใช้ Transcript Hint แล้วหักคะแนนเล็กน้อย
5. หลัง Submit: เปิด Full Transcript เพื่อ review

## Logs
- transcriptHints
- student_id
- session
- type
- timestamp

## เหตุผลเชิงการสอน
ช่วยรักษาความเป็น listening task และยังให้ transcript เพื่อเรียนรู้หลังทำงานเสร็จ
