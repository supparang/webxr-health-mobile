# 03 — คู่มือนักศึกษา 1 หน้า

## วิธีเข้าเล่น

1. เปิดลิงก์ที่ครูส่งให้
2. กรอกชื่อ รหัสนักศึกษา และ Section
3. ตรวจ Bank / Mode / Difficulty
4. กด Start
5. เล่นให้จบ 1 รอบ
6. ดู Summary และคำแนะนำหลังเล่น

## โหมดหลัก

| Mode | เหมาะกับ |
|---|---|
| AI Training | ฝึกพื้นฐาน |
| Speed Run | ฝึกความเร็ว |
| Debug Mission | ฝึกใช้คำในบริบท |
| Boss Battle / Boss Rush | ทดสอบความพร้อม |

## AI Help

- ใช้ได้จำกัด
- ใช้แล้วระบบจะบันทึกว่า AI Assisted
- ควรลองตอบเองก่อนเพื่อเพิ่มการเรียนรู้

## หลังเล่น

ถ้าระบบแนะนำ Weak Rescue / Debug Mission ให้ฝึกตามนั้น เพราะเป็นคำหรือทักษะที่ยังอ่อน
