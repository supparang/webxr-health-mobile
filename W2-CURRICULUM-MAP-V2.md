# CSAI2601 Week 2 — Design Thinking Studio Curriculum Map

## Direct source alignment

| Week 2 slide content | W2 game implementation |
|---|---|
| Learning outcomes: explain HCD / Design Thinking, analyze symptoms vs actual user problem, write a Design Challenge | Completion produces a Design Challenge Brief v1 from the learner's decisions |
| Solution-first vs Problem-first | Define phase requires a Problem Statement and HMW; solution-first and broad alternatives are intentionally plausible but not suitable |
| Human-Centered Design: user, context, task, evidence | Empathize Board asks learners to select evidence that explains user goal, context, and friction |
| Empathize → Define → Ideate → Prototype → Test | Five phase rail; choices made in earlier stages alter the Test outcome |
| Iteration is not failure | Sprint Retrospective returns to the weakest phase without revealing an answer key |
| Diverge / Converge | Ideate makes the learner choose the single experiment direction worth testing first |
| Design artifacts | Evidence Notes, Problem Statement, HMW, Prototype Plan, Test Learning, Brief v1 |
| Week 2 Project Workshop and rubric | Project Gate checks a brief that names the user/context/evidence/unknown and a credible next step |

## Bank structure

24 challenges: 8 real-use service contexts × 3 variants.

- Advisor Appointment
- Scholarship Portal
- Course Registration
- Library Digital Resources
- Campus Repair Service
- Campus Clinic Booking
- Study Room Booking
- Campus Shuttle

Variants change user constraints and reasoning context:

1. First use
2. Mobile / time pressure
3. Low confidence / error concern

## Assessment model

The studio assesses a reasoning chain, not memorized terminology:

`Evidence → Problem Statement + HMW → Experiment direction → Prototype → Test decision → Project Brief`

- 1★: Practice Sprint Clear
- 2★: Project Brief Ready (Evidence ≥ 2/3, Define correct direction, Test + Project Gate aligned, Sprint Energy ≥ 70)
- 3★: Evidence-led Sprint Mastery (all phases coherent, 3/3 evidence, energy ≥ 85)
