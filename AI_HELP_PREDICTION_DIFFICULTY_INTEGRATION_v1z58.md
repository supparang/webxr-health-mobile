# v1z58 AI Help + Prediction + Difficulty Integration

AI Learning Layer:
1. AI Help
   - scaffold only, not answer key
   - hint / frame / keyword / checklist
   - adaptive limit by skill, difficulty, and session level

2. AI Prediction
   - readiness score 0-100
   - predicts Ready / Almost Ready / Needs Practice / High Risk
   - considers scores, passed skills, stars, and AI usage

3. AI Difficulty
   - recommends easy / normal / hard / challenge
   - high readiness -> harder replay
   - low readiness -> scaffold practice

Important:
High AI help usage should trigger independent replay before Boss Gate confidence is considered stable.
