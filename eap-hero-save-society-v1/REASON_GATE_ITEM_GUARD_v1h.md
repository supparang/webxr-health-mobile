# REASON_GATE_ITEM_GUARD_v1h

## เพิ่มอะไร
1. Reason Gate / Justification Attack
   - เมื่อผู้เล่นตอบถูกใน Boss Battle ระบบอาจถามเหตุผลต่อ
   - ถ้าเหตุผลถูก จะโจมตีบอสแรงขึ้น
   - ถ้าเหตุผลผิด จะโจมตีได้แค่บางส่วน
   - เปิดบ่อยขึ้นใน Hero / No Hint / Speed Contract, Rage Mode และ combo สูง

2. Item Guard Dashboard
   - ตรวจ Answer Longest Rate
   - ตรวจ Average Length Gap
   - ตรวจ Similar Question Groups
   - ใช้ดูว่า Session ไหนควรปรับคลังข้อเพิ่มก่อนสอบจริง

## ทำไมสำคัญ
การลด length cue ด้วยตัวเลือกอย่างเดียวอาจไม่พอ
Reason Gate ทำให้ผู้เล่นต้องรู้เหตุผลของคำตอบ จึงลดการเดาได้แรงกว่า
