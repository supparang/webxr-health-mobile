# v1z66 Live Conversation Speaking

## New Speaking Flow
Quick Plan → Round 1 Speak to Human Partner → British Wildcard → Round 2 Respond → Human Stamp + Glow/Grow → Optional Voice Review → Save Evidence

## What was removed from the main experience
- Large transcript-first box
- Student self-checkboxes for speaking quality
- Duplicate Start Speaking / Start Voice Input flow

## Human Stamp
- Clear: clear message, task details and wildcard response
- Developing: communication is emerging; save feedback and practise next time
- Retry: rehearse and answer the live follow-up again

Accent is not graded. Human feedback focuses on clarity, task relevance, evidence and interaction.
