# v1z68 Clean Route Ledger + Map Clarity

## Why this patch exists
Older test evidence from previous game versions remained in Local Storage. It could make later Sessions and Boss Gates look unlocked even though the learner had only just completed S1 in the new UK–Thailand course.

## New rule
Only evidence saved with:
UKTH_GLOBAL_CAMPUS_20260625
counts for the new route.

Existing records are not deleted. They remain available for reports, but they cannot unlock S2–B5.

## Safe transition
When this patch first opens while the learner is on passed S1, the S1 pass is carried into the clean route. The next action becomes:
Continue to S2 · UK Campus Decoder
