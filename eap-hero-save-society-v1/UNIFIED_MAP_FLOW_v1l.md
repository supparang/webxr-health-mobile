# UNIFIED_MAP_FLOW_v1l

## ปัญหาที่แก้
ผู้เล่นอาจรู้สึกว่าเล่นทั้ง Map และ Skills แยกกันเหมือนคนละเกม

## Concept ใหม่
Map เป็นเส้นทางหลักเดียวของเกม
กด Session จาก Map → เข้า Session Path → ทำ Core/Support Skill Mission → ปลด Boss Gate → กลับ Map

## UX Lock
- ไม่ให้ Skills เป็นโหมดหลักแยกใน top nav
- Skill Hub เป็นหน้าภายใน Session Path เท่านั้น
- Replay Hub ยังแยกได้ เพราะเป็น post-learning challenge / replay layer
- Teacher / QA / Exam ยังเป็น utility layer
