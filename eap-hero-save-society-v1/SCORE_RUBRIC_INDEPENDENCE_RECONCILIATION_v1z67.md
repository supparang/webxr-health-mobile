# v1z67 — Score · Rubric · Independence Reconciliation

## What the screenshot revealed
An 87/100 Excellent task score sat beside an AI rubric score and “Needs Independent Replay”.
That looked like two conflicting grades.

## New report structure
- Mission Task Score = game result
- Task Checklist = formative evidence against the exact task; not a second grade
- Independence Status = whether AI support was used; it does not cancel the mission score

## Gold Reading A2/A2+ rubric
The checklist now evaluates the exact three visible answers:
1. Main Point
2. Key Phrase
3. Direct Detail
4. Clear completion

It no longer asks S1 Reading learners for an unrelated critical-reading check.

## Supported attempts
When two or more helps were used, the report now says:
“Supported practice · independent replay available”
The score remains valid. An independent replay is for confirming no-help evidence later.

Existing browser reports are recalculated automatically on first v1z67 load.
