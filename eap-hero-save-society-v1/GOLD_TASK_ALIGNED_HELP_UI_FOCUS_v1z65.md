# v1z65 — Gold Task-Aligned Help + UI Focus

## Screenshot issue fixed
The previous generic Aligned Help frames could mismatch the active Gold source. For example, an Academic Goal task could show an old frame such as “This passage is for ___.”

## New rule
Gold authored source → source-specific help card → source-aware AI Mentor message.

The system now:
- registers the exact active Gold source for Reading, Writing, Listening, and Speaking;
- renders task-specific steps and sentence frames by skill and A2/A2+/B1/B1+ tier;
- makes AI Help refer to the current Gold source without giving the answer;
- removes duplicate Gold-source wording from the AI Ability card, leaving source identity in one compact Gold badge;
- exposes an alignment audit with 15 × 4 skills × 4 AI tiers = 240 Gold task configurations.

## Example: S1 Weekly Goal Tracker, Reading A2
Frames are now:
- The main point is that ___.
- One key phrase is “___.”
- The passage says that ___.

They no longer ask about a reader group or passage purpose.
