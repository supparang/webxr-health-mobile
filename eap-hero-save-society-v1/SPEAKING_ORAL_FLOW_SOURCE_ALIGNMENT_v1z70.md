## New oral flow
1. Start Speaking
   - Starts the timer.
   - Starts live transcript automatically when supported.
2. Finish Speaking
   - Stops timer and live transcript.
3. Confirm four actual oral elements
   - target time
   - topic/opening
   - one source detail
   - clear closing
4. Submit
   - transcript/notes are optional evidence for more detailed feedback.

## Scoring
The Speaking mission score is oral-first:
- target duration
- learner-confirmed oral attempt
- topic/opening
- source detail
- closing
- optional signpost
- optional transcript contributes only a small language-feedback component

A2 Foundation is 25–35 seconds and requires topic + one source detail + closing.
