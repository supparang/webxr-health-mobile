# RUBRIC_REVIEW_MODE_v1r

## เป้าหมาย
ทำให้ EAP Hero ใช้ตรวจงาน Reading/Writing/Listening/Speaking ได้จริง ไม่ใช่มีแต่ auto-score

## เพิ่ม
1. Portfolio Review Queue
2. Rubric Review ตาม skill
3. Teacher Feedback
4. Review Status
   - Reviewed
   - Needs Revision
   - Excellent
   - Needs Support
5. Export Rubric CSV
6. Firebase preview สำหรับ rubricReviews

## Rubric
Writing:
Task Achievement, Organization, Academic Tone, Vocabulary, Grammar, Citation/AI Ethics

Speaking:
Opening, Organization, Signposting, Evidence, Fluency, Q&A

Reading:
Main Idea, Keywords, Evidence, Inference, Critical Check, Clarity

Listening:
Main Point, Keywords, Signal Words, Evidence Note, Summary, Follow-up Question
