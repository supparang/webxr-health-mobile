UX Detective

HOW TO PLAY

คิดจากคนใช้ ไม่ใช่จากหน้าจอ

  1. อ่าน User Goal และคำพูดของผู้ใช้
  2. เลือกจุดที่น่าจะเป็นต้นเหตุของความติดขัด
  3. วิเคราะห์ความสัมพันธ์ระหว่าง UI Symptom และ UX Impact
  4. เลือกวิธีแก้ที่ช่วยผู้ใช้ทำงานสำเร็จมากที่สุด
  5. อ่านผล User Test และอธิบายเหตุผลด้วยคำของคุณเอง

ผ่านเมื่อมีคะแนนอย่างน้อย 60 และจบ Explain Check ครบทุกเคส คะแนนเน้นเหตุผล ไม่เน้นกดเร็ว
