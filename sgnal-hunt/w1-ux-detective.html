HOW TO PLAY W1

หลักยังเหมือนเดิม แต่รูปแบบภารกิจเปลี่ยนจริง

  1. Evidence Triage — ตัดสินใจว่า evidence ใดขวาง User Goal ก่อน
  2. Goal Route — หาเส้นทางเริ่มงานที่สอดคล้องกับเป้าหมาย
  3. UI / UX Split — แยก symptom บนจอออกจาก impact ต่อผู้ใช้
  4. Budget Trade-off — เลือกการแก้ภายใต้ Design Energy จำกัด
  5. A/B Audit และ Test Forecast — เปรียบเทียบแบบและทำนายผล User Test
